# CuriousBees V2 — Dashboard Completion Report

This report documents the verification and completion state of the three target portals: **Research Scholar**, **Faculty Supervisor**, and **Institution Admin**.

---

## 🧑‍🎓 1. Research Scholar Dashboard

Designed for PhD scholars to track their research activities, deadlines, and logs.

### 📋 Feature Checklist

| Subview / Module | Status | Verification Details |
| :--- | :---: | :--- |
| **Overview / Metrics** | `COMPLETED` | Displays academic H-Index, total publications count, active workspaces progress bar, and calendar widgets. |
| **Publications Registry** | `COMPLETED` | View registered journals/DOIs. Link to log new publications. |
| **Research Progress Logs** | `COMPLETED` | Submit periodic progress logs, view list of submitted reports and their approval state. |
| **Events Timeline** | `COMPLETED` | Grid showing upcoming deadlines, Gantt chart milestone deadlines, and monthly calendar dates. |
| **Academic Opportunities** | `COMPLETED` | Vacancy board listing supervisor research projects. Quick action to submit applications. |
| **Discussion Threads** | `COMPLETED` | Academic forum to start and read threads. |
| **Workspaces Hub** | `COMPLETED` | List active collaboration sandboxes with milestone tracking and files registry. |
| **Activity Notifications** | `COMPLETED` | Push broadcast log alerts feed showing project updates. |

---

## 👩‍🏫 2. Faculty Supervisor Dashboard

Allows supervisors to track and review their mapped PhD research scholars.

### 📋 Feature Checklist

| Subview / Module | Status | Verification Details |
| :--- | :---: | :--- |
| **Scholar Management** | `COMPLETED` | Lists all assigned scholars. Accessible via `/my-scholars` portal page. |
| **Student Tracking** | `COMPLETED` | View progress charts, check logged publications, and view individual scholar milestones. |
| **Progress Monitoring** | `COMPLETED` | Check active workspaces progress percentage bars and timelines. |
| **Approvals & Grading** | `COMPLETED` | Grade and review scholar progress logs. Toggle approval status (Approved, Rejected, Pending). |
| **Quick Actions** | `COMPLETED` | Post research opportunities, schedule calendar events, and broadcast notices. |

---

## 👑 3. Institution Admin Dashboard

Designed for university node administrators to manage global parameters. This view is restricted to system settings and contains no researcher collaboration widgets.

### 📋 Feature Checklist

| Subview / Module | Status | Verification Details |
| :--- | :---: | :--- |
| **User Monitoring** | `COMPLETED` | Audit all registered profiles, roles, and approval status. Accessible under `/admin/users`. |
| **Analytics Board** | `COMPLETED` | Global department research counts and citation analytics under `/admin/analytics`. |
| **Department Registry** | `COMPLETED` | Add and configure campus departments (e.g. CSE, Biotechnology) under `/admin/departments`. |
| **System Diagnostics** | `COMPLETED` | Track runtime Node engine status, physical memory usage, and load telemetry under `/admin/settings`. |
| **Reports Audit** | `COMPLETED` | Access and review global progress logs and department activity audits. |
